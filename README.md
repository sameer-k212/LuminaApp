# LuminaApp

Learning platform with course management and content highlighting.

## Deploy on Netlify

1. Push to GitHub
2. Import to Netlify
3. Add environment variables (see netlify.toml)
4. Deploy

## Local Development

```bash
# Backend
cd backend
npm install
npm start

# Frontend
cd frontend
npm install
npm run dev
```

## Tech Stack

- Frontend: React + Vite + Tailwind
- Backend: Express + MongoDB
- Storage: Cloudinary
